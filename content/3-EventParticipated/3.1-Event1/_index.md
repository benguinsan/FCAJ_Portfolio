# Event Report: FCAJ Community Day

### Event Information
- **Event:** FCAJ Community Day
- **Date & Time:** Saturday, March 21, 09:00 - 12:00
- **Location:** AWS Vietnam Office (Floors 26 & 36), Ho Chi Minh City
- **Role:** Attendee (FCAJ Cloud Intern)

### Event Overview
This event provided opportunities to connect with FCAJ community members, learn directly from industry experts, and explore current discussions around cloud technology and innovation.

### Speakers
- Hai Bui
- Phuc Dang
- Phap Nguyen
- Phat Pham
- Nghi Danh
- Phong Nguyen
- Thinh Nguyen

### Session Highlights
1. **Building Modern Platform Engineering & Career Pathways**
   - Company culture and internship opportunities.
   - Introduction to Platform Engineering in modern Cloud/DevOps environments.
   - Interactive Q&A with speakers.

2. **GenAIOps Essential - DevOps for Generative AI Applications**
   - DevOps fundamentals on AWS and learning resources.
   - Real use cases with Bedrock AgentCore Observability, EKS, and Langfuse.

3. **Shipping Code in the Agentic Era**
   - Core problem context and practical tooling.
   - Deep dive, productivity playbook, and live demo.

4. **Production-Grade Multimodal GenAI on AWS**
   - New AI application stack.
   - Multimodal search with Nova Embeddings.
   - GraphRAG for enterprise knowledge.
   - Multi-agent workflows.
   - Safe and observable GenAI practices.

5. **Amazon CloudFront for Every Workload**
   - Cost optimization.
   - Security capabilities.
   - Reliability improvements.
   - Performance enhancements.

### Key Takeaways
- Platform Engineering is both a technical foundation and a practical career pathway.
- GenAIOps requires not only model usage but also observability and operational discipline.
- AI-assisted workflows can significantly improve engineering productivity.
- Production AI systems must balance performance, safety, and maintainability.
